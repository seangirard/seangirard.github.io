Spokane Public Library

Internal workflow application

I built a public-facing tool for customers to request titles the library does not currently own (www.spokanelibrary.org/request/).

The material request form was more popular than we anticipated (we went from processing several paper requests per month to well over 1000). So it was clear right away that we would need a better system for processing requests.

  • Streamlined review process with change tracking
  • Automated customer notification
  • ILS integration for checking status and placing holds

this app is behind a staff login

Spokane Public Library Newsletter

Email newsletter publishing and delivery

The Library's previous newsletter platform was stuggling to keep up with a list of over 70,000 and offered no delivery tracking or analytics.

Commercial platforms were cost prohibitive for our list size, so I built a custom platform using WordPress and Mailgun.

  • Developed Wordpress plugin to compose and publish emails
  • Designed email templates based on Zurb Ink
  • Integrated with Mailgun API for list and campaign management

sign up for the newsletter: news.spokanelibrary.org/subscribe/

Spokane Business Library

Joint organization website

Spokane Public Library, the Spokane County Library District, and other regional library have a large number of subscription data sources available for the business community.

SPL and SCLD felt they had a communication problem with the public regarding what is available and who it is available to – different services are available to different library customers.

  • Designed a website to showcase business resources and steer customers toward the appropriate venue
  • Worked closely with stakeholders from SPL, SCLD, and EWU.
  • Integrated RSS feeds from multiples sources
  • Built the website using Jekyll to eliminate server side dependencies

visit the business library: www.spokanebusiness.org
